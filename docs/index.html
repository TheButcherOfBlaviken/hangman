This is the documentation for the Hangman game that I have developed. Rules of hangman are pretty simple, we have all played this game when we were younger.

The entire game is one big class with multiple methods. The method game.Hangman.hangman() contains the whole game. Multiple static and callable methods are used throughout the game.Hangman.hangman() method in order to simplify the main function that really runs the game.
